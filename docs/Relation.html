Clase Relation

Define vínculos direccionales y semánticos entre diferentes entidades del sistema.

Definición

La clase Relation permite establecer un puente entre dos objetos identificables, especificando la naturaleza de su conexión mediante un tipo definido.

Atributos de Datos

Atributo Tipo Descripción
from_id str El identificador del objeto de origen (sujeto).
to_id str El identificador del objeto de destino (objeto).
connection_type str Cadena que describe la lógica de la unión (ej. "referencia", "depende de").

Notas de Implementación

A diferencia de Entry, esta clase es una representación pura de datos (Data Transfer Object) diseñada para grafos de conocimiento o trazabilidad de dependencias.
